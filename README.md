## 🚀 What I've Built
 
### 🔐 UPI Offline Mesh — Decentralized Payment Settlement
`Java 17` `Spring Boot` `JUnit` `H2 Database` `REST`
 
Offline P2P payment system that doesn't need internet. Uses hybrid cryptography (RSA-OAEP + AES-256-GCM) so payloads are tamper-proof across distributed nodes. Built idempotency with `ConcurrentHashMap.putIfAbsent` to kill duplicate transactions.
 
[![View Repo](https://img.shields.io/badge/View_Repo-181717?style=flat-square&logo=github)](https://github.com/bmw-hub/UPI_Without_Internet)
 
---
 
### 🔗 URL Shortener Service
`Java 11` `Spring Boot` `PostgreSQL` `JPA/Hibernate` `REST APIs`
 
Clean REST backend with Base62 short-code generation, full CRUD, and global exception handling. Follows SOLID strictly — every layer is testable and separated.
 
[![View Repo](https://img.shields.io/badge/View_Repo-181717?style=flat-square&logo=github)](https://github.com/bmw-hub/UrlShortener)

### 🔍 Forensic Face Sketch Recognition
`Java` `AWS Rekognition` `AWS S3` `JavaFX` `REST APIs`
 
Desktop app that takes a hand-drawn forensic sketch, sends it to AWS Rekognition, and finds matching faces in a database. OTP auth + role-based access + multi-threaded S3 sync built in.
